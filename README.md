# Walkie Talkie Node (2023)

Simple Socket.IO messages for Walkie Talkie project.

This project was made with Node.js/Express in 2023.

## Getting started

To run the project:
1. Use `npm ci` to install dependencies.
2. Start the project with `npm run dev`
